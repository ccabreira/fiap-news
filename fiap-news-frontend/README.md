# FIAP News - Frontend

Este projeto é um frontend React integrado a uma API para exibição e gerenciamento de notícias.

## Tecnologias utilizadas:
- React.js
- Vite
- React Router DOM
- Consumo de API via Fetch
- Autenticação com JWT
